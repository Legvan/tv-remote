Privacy Policy

App: TV Remote  |  Developer: Leszek Pokorski  |  Last updated: April 2026

Overview

TV Remote is a local-network tool that lets any browser on your home network control an Android TV. This policy explains what data the app does and does not collect.

Data Collection

We collect no personal data. The app does not collect, store, transmit, or share any personal information, usage statistics, analytics, or identifiers.

Network Communication

  • The app runs an HTTP server on port 8080 that is accessible only to devices on the same local network (LAN). It is not reachable from the internet.
  • The app connects to the device's own ADB daemon at 127.0.0.1:5555 (loopback) to inject remote control commands. This connection never leaves the device.
  • No data is sent to any external server, cloud service, or third party.

Local Storage

The app generates an RSA key pair the first time it runs. This key pair is stored in the app's private internal storage and is used solely to authenticate the loopback ADB connection. It is never transmitted off the device.

Permissions

  • INTERNET — required to bind the local HTTP server socket and the loopback ADB socket.
  • ACCESS_NETWORK_STATE / ACCESS_WIFI_STATE — used to read the device's local IP address for display in the UI.
  • WAKE_LOCK — keeps the CPU active while the server is running so commands are not dropped.
  • FOREGROUND_SERVICE / FOREGROUND_SERVICE_CONNECTED_DEVICE — required to run the persistent server as a foreground service with a visible notification.
  • RECEIVE_BOOT_COMPLETED — allows the server to auto-start after a device reboot.

Third-Party Services

The app uses no third-party SDKs, analytics libraries, advertising frameworks, or crash-reporting services.

ADB Requirement

The app requires the user to explicitly enable USB Debugging / ADB over network in the device's Developer Options. This is a deliberate, non-default setting. The app only interacts with the device's own ADB daemon and does not connect to any external ADB host.

Children's Privacy

The app does not collect any data from any user, including children. It is not directed at children.

Changes to This Policy

If the app's data practices change, this page will be updated with a new date. Given the app's local-only design, no material changes are anticipated.
